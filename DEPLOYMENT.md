# 🚀 Panduan Deployment Website

Berikut adalah beberapa cara untuk meng-online-kan website personal branding Anda:

## 1. GitHub Pages (GRATIS & MUDAH) ⭐ Recommended

### Langkah-langkah:

1. **Buat akun GitHub** di https://github.com (jika belum punya)

2. **Buat repository baru**:
   - Klik tombol "New" atau "+"
   - Nama repository: `username.github.io` (ganti username dengan username GitHub Anda)
   - Pilih "Public"
   - Klik "Create repository"

3. **Upload file**:
   - Klik "uploading an existing file"
   - Drag & drop semua file website (index.html, about.html, dll)
   - Klik "Commit changes"

4. **Aktifkan GitHub Pages**:
   - Masuk ke Settings repository
   - Scroll ke bagian "Pages"
   - Source: pilih "main" branch
   - Klik "Save"

5. **Akses website**:
   - Website akan tersedia di: `https://username.github.io`
   - Tunggu 1-5 menit untuk deployment pertama

### Kelebihan:
✅ Gratis selamanya
✅ Mudah digunakan
✅ SSL/HTTPS otomatis
✅ Custom domain support

---

## 2. Netlify (GRATIS) ⭐ Recommended

### Langkah-langkah:

1. **Buat akun** di https://netlify.com

2. **Deploy website**:
   - Klik "Add new site" → "Deploy manually"
   - Drag & drop folder website Anda
   - Tunggu proses upload selesai

3. **Website langsung online!**
   - URL: `random-name.netlify.app`
   - Bisa custom domain

### Kelebihan:
✅ Gratis
✅ Sangat cepat
✅ Drag & drop deployment
✅ Auto SSL
✅ Custom domain gratis

---

## 3. Vercel (GRATIS)

### Langkah-langkah:

1. **Buat akun** di https://vercel.com

2. **Import project**:
   - Klik "New Project"
   - Upload folder atau connect GitHub
   - Klik "Deploy"

3. **Website online!**
   - URL: `project-name.vercel.app`

### Kelebihan:
✅ Gratis
✅ Deployment cepat
✅ Analytics gratis
✅ Custom domain

---

## 4. InfinityFree (GRATIS)

### Langkah-langkah:

1. **Buat akun** di https://infinityfree.net

2. **Buat hosting account**:
   - Pilih subdomain atau custom domain
   - Tunggu aktivasi (1-24 jam)

3. **Upload file via FTP**:
   - Download FileZilla
   - Connect ke FTP (info di email)
   - Upload semua file ke folder `htdocs`

### Kelebihan:
✅ Gratis
✅ Unlimited bandwidth
✅ Support PHP & MySQL
✅ cPanel

---

## 5. 000webhost (GRATIS)

### Langkah-langkah:

1. **Buat akun** di https://www.000webhost.com

2. **Buat website**:
   - Klik "Create Website"
   - Pilih nama website

3. **Upload file**:
   - Gunakan File Manager
   - Upload ke folder `public_html`

### Kelebihan:
✅ Gratis
✅ 300 MB storage
✅ No ads
✅ Easy file manager

---

## 6. Tiiny.host (GRATIS untuk 7 hari)

### Langkah-langkah:

1. **Buka** https://tiiny.host

2. **Upload ZIP**:
   - Zip semua file website
   - Drag & drop ke website
   - Tunggu upload selesai

3. **Website langsung online!**

### Kelebihan:
✅ Tidak perlu daftar
✅ Instant deployment
✅ Gratis 7 hari

## 📝 Tips Deployment

### Sebelum Deploy:
- ✅ Test website di local browser
- ✅ Pastikan semua link berfungsi
- ✅ Cek responsive di mobile
- ✅ Pastikan gambar ter-load

### Setelah Deploy:
- ✅ Test di berbagai browser
- ✅ Test di mobile device
- ✅ Cek loading speed
- ✅ Share link ke teman untuk feedback

### Custom Domain (Opsional):
Jika ingin domain sendiri (contoh: galihseto.com):
1. Beli domain di Namecheap, GoDaddy, atau Niagahoster
2. Connect domain ke hosting (ikuti panduan masing-masing platform)
3. Tunggu DNS propagation (1-48 jam)

## 🆘 Troubleshooting

### Website tidak muncul?
- Tunggu 5-10 menit setelah deploy
- Clear browser cache (Ctrl + Shift + Delete)
- Coba browser lain

### Gambar tidak muncul?
- Pastikan URL gambar benar
- Cek koneksi internet
- Pastikan gambar di Imgur masih aktif

### CSS tidak ter-load?
- Pastikan file styles.css ada di folder yang sama
- Cek nama file (case-sensitive)
- Clear cache browser
