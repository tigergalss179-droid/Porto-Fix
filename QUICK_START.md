# 🚀 Quick Start Guide

Panduan cepat untuk memulai menggunakan website personal branding Anda!

---

## ⚡ 3 Langkah Mudah

### 1️⃣ Buka Website
- Double-click file `index.html`
- Website akan terbuka di browser default Anda
- Navigasi menggunakan menu di atas

### 2️⃣ Kustomisasi (Opsional)
- Edit informasi pribadi di file HTML
- Ubah warna di `styles.css`
- Ganti foto profil dengan URL Anda
- Lihat panduan lengkap di `CUSTOMIZATION.md`

### 3️⃣ Deploy Online
- Pilih platform hosting (GitHub Pages, Netlify, dll)
- Upload semua file
- Share link website Anda!
- Lihat panduan lengkap di `DEPLOYMENT.md`

---

## ✏️ Edit Cepat

### Mengubah Nama & Info Dasar

**File: `index.html`**
```html
<!-- Baris ~48 -->
<h1 class="fade-in">NAMA ANDA</h1>

<!-- Baris ~49 -->
<p class="tagline fade-in-delay">TAGLINE ANDA</p>

<!-- Baris ~50-53 -->
<p class="description fade-in-delay-2">
    DESKRIPSI DIRI ANDA
</p>
```

### Mengubah Foto

**File: `index.html` dan `cv.html`**
```html
<!-- Ganti URL ini -->
<img src="https://i.imgur.com/mcW08Ge.jpg" alt="Nama Anda">

<!-- Dengan URL foto Anda dari Imgur -->
<img src="https://i.imgur.com/FOTO_ANDA.jpg" alt="Nama Anda">
```

### Mengubah Warna Tema

**File: `styles.css` (baris 13-16)**
```css
:root {
    --gradient-start: #5D69E2;  /* Warna awal */
    --gradient-mid: #7B5BB1;    /* Warna tengah */
    --gradient-end: #8C4A9E;    /* Warna akhir */
}
```

## 🌐 Deploy ke GitHub Pages (5 Menit)

1. **Buat akun GitHub**: https://github.com
2. **Buat repository baru**: `username.github.io`
3. **Upload semua file** ke repository
4. **Tunggu 2-5 menit**
5. **Buka**: `https://username.github.io`

✅ **DONE!** Website Anda sudah online!

## 📱 Test Responsive

Buka website di browser, tekan:
- **F12** untuk buka Developer Tools
- **Ctrl + Shift + M** untuk toggle device toolbar
- Test di berbagai ukuran layar

Atau test langsung di:
- Desktop/Laptop
- Tablet
- Smartphone
