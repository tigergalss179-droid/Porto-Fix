# Website Personal Branding - Muhammad Galih Seto

Website personal branding profesional dengan tema minimalist dan interaktif.

## 🎨 Fitur Utama

- **Design Minimalist Profesional** - Tampilan bersih dan modern
- **Gradient Header** - Warna gradient indah (#5D69E2 → #7B5BB1 → #8C4A9E)
- **Animasi Interaktif** - Smooth animations dan hover effects
- **Fully Responsive** - Tampil sempurna di semua perangkat
- **5 Halaman Lengkap** - Home, About, Skills, Portfolio, CV

## 📄 Struktur Halaman

### 1. Homepage (index.html)
- Nama dan foto profil
- Tagline: "Interested in CyberSec and Networking"
- Deskripsi diri
- Quick info cards
- Social media links

### 2. About Me (about.html)
- Riwayat keluarga
- Riwayat pendidikan dengan timeline
- Hobi dengan icon interaktif
- Minat dan cita-cita
- Soft skills & Hard skills

### 3. Skills (skills.html)
- Progress bar untuk setiap skill:
  - HTML: 20%
  - CSS: 20%
  - JavaScript: 10%
  - Cisco Packet Tracer: 70%
  - Editing (Canva): 75%
  - Networking: 50%
- Animasi progress bar saat scroll

### 4. Portfolio (portfolio.html)
- **Desain Mug** - Project desain grafis
- **Website Kelompok** - Game journal review website
- Hover effects dan overlay
- Tags untuk setiap project

### 5. CV (cv.html)
- Layout 2 kolom profesional
- Foto profil
- Data pribadi dan kontak
- Skills dengan progress bar
- Riwayat pendidikan timeline
- Quote motivasi
- Tombol download/print CV

## 🚀 Cara Menggunakan

1. **Download semua file** ke satu folder:
   - index.html
   - about.html
   - skills.html
   - portfolio.html
   - cv.html
   - styles.css
   - script.js

2. **Buka index.html** di browser (Chrome, Firefox, Edge, dll)

3. **Navigasi** menggunakan menu di atas

## 🎨 Kustomisasi

### Mengubah Warna
Edit file `styles.css` bagian `:root`:
```css
:root {
    --gradient-start: #5D69E2;
    --gradient-mid: #7B5BB1;
    --gradient-end: #8C4A9E;
}
```

### Mengubah Foto
Ganti URL foto di file HTML:
```html
<img src="https://i.imgur.com/mcW08Ge.jpg" alt="Muhammad Galih Seto">
```

### Mengubah Konten
Edit langsung teks di file HTML sesuai kebutuhan.

## 📱 Responsive Design

Website otomatis menyesuaikan tampilan untuk:
- Desktop (> 968px)
- Tablet (600px - 968px)
- Mobile (< 600px)

## ✨ Fitur Interaktif

- **Smooth Scrolling** - Navigasi halus
- **Hover Effects** - Animasi saat hover
- **Progress Bar Animation** - Animasi skill bars
- **Mobile Menu** - Hamburger menu untuk mobile
- **Scroll to Top Button** - Tombol kembali ke atas
- **Parallax Effect** - Efek parallax pada hero image
- **Typing Effect** - Efek mengetik pada tagline
- **Ripple Effect** - Efek ripple pada tombol
- **Gallery Modal** - Galeri gambar untuk portfolio (NEW!)
- **Lightbox** - Zoom gambar full screen (NEW!)

## 🖨️ Print CV

Klik tombol "Download CV (PDF)" di halaman CV atau tekan `Ctrl+P` untuk print.
