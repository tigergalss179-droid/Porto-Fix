## 🎬 Quick Start (Super Cepat!)

Tidak mau baca panduan? Ikuti ini:

### 1. Edit Informasi (5 menit)
```
1. Buka index.html dengan Notepad/VS Code
2. Cari "Muhammad Galih Seto" → Ganti dengan nama Anda
3. Cari "Interested in CyberSec" → Ganti dengan tagline Anda
4. Save (Ctrl + S)
5. Refresh browser (F5)
```

### 2. Ganti Foto (5 menit)
```
1. Upload foto ke imgur.com
2. Copy link foto (harus ada i. dan .jpg)
3. Buka index.html
4. Cari "https://i.imgur.com/mcW08Ge.jpg"
5. Ganti dengan link foto Anda
6. Save dan refresh
```

### 3. Upload Online (10 menit)
```
1. Buka github.com
2. Sign up (jika belum punya akun)
3. Buat repository baru: username.github.io
4. Upload semua file website
5. Tunggu 5 menit
6. Buka: https://username.github.io
7. DONE! 🎉
```

## 💡 Tips Penting

### ✅ Do's (Lakukan):
- Backup file sebelum edit
- Test setiap perubahan

- Share link website di CV dan social media

### ❌ Don'ts (Jangan):

## 🆘 Butuh Bantuan?

### Masalah Umum:

**Website tidak muncul?**
→ Pastikan file index.html ada di folder yang sama dengan file lain

**Foto tidak muncul?**
→ Cek link Imgur (harus ada `i.` dan ekstensi `.jpg`)

**Perubahan tidak terlihat?**
→ Save file (Ctrl+S) dan refresh browser (Ctrl+F5)

**CSS berantakan?**
→ Jangan edit `styles.css` jika tidak paham CSS

**Masih bingung?**
→ Baca `PANDUAN_LENGKAP.md` bagian FAQ
