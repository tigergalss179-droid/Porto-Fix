## 💡 How to Report Issues

Jika menemukan bug atau masalah:

1. **Check Known Issues** di atas
2. **Check FAQ** di PANDUAN_LENGKAP.md
3. **Test di browser lain** (Chrome, Firefox, Edge)
4. **Clear cache** (Ctrl + Shift + Delete)
5. **Check console** (F12) untuk error

## 📝 Update Notes

### How to Update

Jika ada update baru:

1. **Backup** file lama
2. **Download** file baru
3. **Replace** file yang diupdate
4. **Test** di local browser
5. **Deploy** jika tidak ada masalah

### Files to Keep (Don't Replace)
- Gambar/foto Anda
- Konten yang sudah dikustomisasi
- Custom CSS/JS yang Anda tambahkan

### Files to Replace (Safe to Update)
- styles.css (jika tidak ada custom CSS)
- script.js (jika tidak ada custom JS)
- Documentation files (.md)
